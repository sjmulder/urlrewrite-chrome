  1. In Source URL, enter the URL that should be rewritten, e.g. http://jira.example.com/
  2. In Destination URL, enter the URL that should replace the Source URL, e.g. http://example.myvpn.com/jira/
  3. Now, any link that begins with Source URL will have that replaced with Destination URL. In the example above, http://jira.example.com/PROJ-123 would be rewritten to http://example.myvpn.com/jira/PROJ-123
